## TreeletHub 1.2.3 for macOS（DMG）

### 安装

1. 下载 `TreeletHub.dmg` 并打开。
2. 将 **TreeletHub** 拖入「应用程序」文件夹。
3. 首次打开若提示无法验证开发者：前往「系统设置 → 隐私与安全性」允许，或右键 App 选择「打开」。
4. 允许 **本地网络** 访问，以便与 iPhone 配对。

### 说明

- 本 DMG 为 Mac 桌面端直装包；iPhone 遥控端请从 App Store 安装。
- 与 [Mac App Store 版](https://apps.apple.com/app/treelethub-%E6%A1%8C%E9%9D%A2%E6%95%88%E7%8E%87%E6%8E%A7%E5%88%B6%E4%B8%AD%E5%BF%83/id6762348258) 二选一即可；App Store 版支持自动更新。

### 系统要求

- macOS 13 或更高版本（以你打包时的部署目标为准）

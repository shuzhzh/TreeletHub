## TreeletHub 1.2.4 for macOS（DMG）

### 新功能：键盘启动器（订阅）

Mac 端新增 **键盘启动器** —— 把常用应用映射到屏幕键盘，用 Control 键一键唤出，比程序坞更顺手的肌肉记忆启动方式。

- **Control 唤出** — 按下并松开 Control，屏幕中央弹出键盘浮层；点击外部、按 Esc 或启动应用后自动关闭。
- **Control + 按键** — 在任意位置快速唤起或隐藏已映射的应用，无需先打开浮层。
- **智能默认映射** — 首次开启按应用名称首字母自动填充键位；支持点击添加/替换、拖拽交换。
- **订阅功能** — 与多页九宫格、灵动岛一样，需有效订阅；过期后开关自动关闭。

首次开启需在「系统设置 → 隐私与安全性 → **输入监控**」中允许 TreeletHub（仅监听 Control 与按键，不记录或上传按键内容）。

### 安装

1. 下载 `TreeletHub.dmg` 并打开。
2. 将 **TreeletHub** 拖入「应用程序」文件夹。
3. 首次打开若提示无法验证开发者：前往「系统设置 → 隐私与安全性」允许，或右键 App 选择「打开」。
4. 允许 **本地网络** 访问，以便与手机配对。

# 常见问题（中文）

[← 返回 README](../README.md)

## 需要账号吗？

不需要。核心配对与九宫格同步通过 **局域网** 完成，不依赖 TreeletHub 云端账号。

## iPhone 找不到 Mac？

1. 确认 Mac 与 iPhone 在 **同一 Wi‑Fi**（避免 guest 网络隔离）。
2. iPhone 是否已允许 **本地网络**（设置 → TreeletHub）。
3. Mac 上 TreeletHub 是否在运行，配对码是否仍有效。
4. 若 Mac 上 **重新生成了配对码**，请在 iPhone 用新码重连。

## Mac 与 iPhone 都要装吗？

是的。Mac 端负责配置九宫格与接收指令；iPhone 端是遥控。请分别从 [Mac App Store](https://apps.apple.com/app/treelethub-%E6%A1%8C%E9%9D%A2%E6%95%88%E7%8E%87%E6%8E%A7%E5%88%B6%E4%B8%AD%E5%BF%83/id6762348258) 与 [iOS App Store](https://apps.apple.com/app/treelethub-%E6%95%88%E7%8E%87-%E6%8E%A7%E5%88%B6%E5%8F%B0/id6762348247) 安装（或使用本仓库 Release 中的 Mac DMG）。

## DMG 和 Mac App Store 版本有什么区别？

功能以各自构建为准。DMG 适合内测或无法使用 App Store 的环境；日常用户建议使用 **Mac App Store** 以获得自动更新与标准签名。

## 订阅在 iPhone 买了，Mac 还要买吗？

Apple 的订阅按 **平台与产品 ID** 计费，Mac 与 iOS 可能需分别购买。以各端「订阅」界面与 App Store 说明为准；Mac 订阅状态会同步到已配对的 iPhone 以解锁多页等能力。

## 数据会上传到你们服务器吗？

配对、布局同步与遥控默认 **仅在局域网**。完整说明见 [隐私政策](https://treelet.us/treelethub/treelet-hub-privacy.html)。天气等功能会按政策访问 Apple 服务。

## 如何反馈问题？

请在本仓库 [Issues](https://github.com/shuzhzh/TreeletHub/issues) 提交，并说明 macOS / iOS 版本与复现步骤。

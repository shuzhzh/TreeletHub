## TreeletHub 是什么？

TreeletHub 让你在 **Mac** 上把常用应用、网页入口与快捷操作收进 **可同步的九宫格**；在 **同一 Wi‑Fi** 下的 **iPhone** 上轻点即可远程启动或切换——不必再走回书桌。

它也是 **Dock 的自然延伸**：把高频启动项收进格子后，你甚至可以在 macOS 里 **隐藏 Dock** 以扩大工作区，常用入口仍在 Mac 与手机上。

- 六位配对码连接，核心同步 **无需云端账号**
- Mac 与 iPhone **两端拖拽** 调整格子顺序
- 订阅可解锁 **多页九宫格** 与 Mac **灵动岛模式** 等能力
- 配对、布局与遥控 **默认仅在局域网** 完成（天气等少数能力会按隐私政策联网）

更完整的营销与功能说明见官网：**[treelet.us/treelethub](https://treelet.us/treelethub/)**（[English](https://treelet.us/treelethub/en/)）

## 三步上手 Quick start

1. 在 **Mac** 打开 TreeletHub，记下界面上的 **6 位配对码**。
2. 在 **iPhone** 打开 TreeletHub →「连接 Mac」，输入配对码并连接；首次请允许 **本地网络**。
3. 在 Mac 九宫格为格子绑定 **应用（.app）** 或快捷项；连接成功后，在 iPhone 的 **Apps** 页轻点即可启动。

详细说明：[docs/USER_GUIDE.zh-CN.md](docs/USER_GUIDE.zh-CN.md) · [docs/USER_GUIDE.en.md](docs/USER_GUIDE.en.md)

## 功能亮点 Highlights

| 能力 | 说明 |
|------|------|
| 九宫格遥控 | 局域网内同步布局，iPhone 一键启动 Mac 应用 |
| 拖拽排序 | Mac 与 iPhone 均可调整格子 |
| 多页扩展 | Apps、Apps1…（部分页面需有效订阅） |
| 显示桌面 | iPhone 已配对时，Apps 页 **双指下滑** 可隐藏全部已打开 Mac 应用并显示桌面 |
| Mac 灵动岛 | 订阅用户可在 Mac 开启顶部浮层：媒体、剪贴板、天气、配对码等（以 Mac 版本为准） |
